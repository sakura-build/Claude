## 1. 快速上手

### 初次使用三步走

```
① 设置昵称 → ② 添加几道菜 → ③ 开始点餐
```

**设置昵称**：点击底部「我的」Tab → 点击「点击设置昵称」→ 输入名字（如「妈妈」「小明」）→ 保存

> 💡 昵称保存在你的手机上，家人各自设置自己的昵称。

## 2. 菜谱管理

### 2.1 添加菜谱

1. 点击底部「**菜谱**」Tab
2. 点击右下角 **+** 按钮
3. 填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 📷 **封面图** | 点击上传成品图 | 手机拍照或从相册选 |
| ✏️ **菜名** | 必填，最多40字 | 西红柿炒鸡蛋 |
| 📂 **分类** | 选择分类 | 家常菜 / 面食 / 汤粥 / 凉菜 / 烘焙 / 小吃 / 其他 |
| 📝 **简介** | 一句话描述 | 国民家常菜，酸甜可口超下饭 |
| ⏱ **烹饪时长** | 滑动选择 | 15 分钟 |
| 📊 **难度** | 点选 | 简单 / 中等 / 困难 |
| 🍽 **几人份** | 点击 +/- 调整 | 2 人份 |
| 🏷 **标签** | 自定义标签 | 快手菜、下饭、减脂 |

4. **添加食材**：在「🥬 食材清单」区域
   - 输入食材名（如"鸡蛋"）
   - 输入数量（如"3"）
   - 选择单位（个/克/毫升/小勺…）
   - 点击「+ 添加食材」继续添加
   - 点击 ✕ 删除某行

5. **添加步骤**：在「👨‍🍳 烹饪步骤」区域
   - 在文本框中描述步骤（如"鸡蛋打散加盐搅匀"）
   - 可选：设置计时（如"炒 2 分钟"）
   - 可选：为这步添加配图（点击「添加步骤图」拍照）
   - 点击「+ 添加步骤」继续添加
   - 点击「删除」移除步骤

6. 点击底部「**保存菜谱**」

### 2.2 浏览与搜索菜谱

- **分类筛选**：点击顶部分类标签（家常菜、面食…）
- **关键词搜索**：在搜索框中输入菜名
- **下拉刷新**：在列表顶部下滑刷新
- **上滑加载**：滑到底部自动加载更多

### 2.3 查看菜谱详情

点击菜谱卡片 → 进入详情页，可以看到：
- 🥬 **食材标签页**：所需食材清单
- 👨‍🍳 **步骤标签页**：分步做法 + 步骤配图 + 计时提醒

### 2.4 编辑与删除

在详情页：
- 点击「✏️ 编辑」→ 修改任意信息后保存
- 点击「删除菜谱」→ 确认后删除（不可恢复）

## 3. 家庭点餐

### 3.1 从菜谱点餐

1. 在菜谱详情页，点击底部「**📝 点这道菜**」
2. 进入点餐页，确认/填写：

| 字段 | 说明 |
|------|------|
| 🍳 **选择菜谱** | 已自动填入，可点击更换 |
| 🕐 **餐别** | 早餐 / 午餐 / 晚餐 |
| 📅 **日期** | 默认明天，可改为任意日期 |
| 🔢 **份数** | 默认 1 份，可加至 10 份 |
| 👤 **点餐人** | 你的昵称，自动填充上次输入 |
| 💬 **备注** | 特殊要求，如「少放盐」「不要辣椒」|

3. 点击「**确认点餐**」

### 3.2 从订单页直接点餐

1. 点击底部「**点餐**」Tab
2. 点击右下角 **+** 按钮
3. 在弹窗中选择菜谱（支持搜索）
4. 填写餐别、日期等信息
5. 确认点餐

## 4. 订单管理

### 4.1 查看订单

点击底部「**点餐**」Tab 查看所有订单列表。

**筛选方式**：
- **按状态**：点击顶部标签「待确认/已接单/制作中/已完成」
- **按日期**：点击「筛选日期」选择特定日期

### 4.2 订单状态流转（厨师操作）

```
家人下单 → [待确认] → 厨师点击「确认接单」
                       ↓
                    [已接单] → 点击「开始制作」
                       ↓
                    [制作中] → 点击「制作完成」
                       ↓
                    [已完成]
```

**如何操作**：
1. 点击订单卡片上的「**更新状态**」按钮
2. 在弹出菜单中选择下一步状态
3. 确认后状态即时更新

**取消订单**：任何非终态订单都可以取消（待确认/已接单/制作中 → 已取消）

### 4.3 首页快速查看

打开小程序首页即可看到：
- 📊 **今日点餐数量**：今天一共点了多少菜
- ⚠️ **待确认数量**：有多少还没确认
- 📋 **今日订单列表**：点击可直接操作状态

## 5. 采购清单

### 5.1 生成清单

当有多道菜被点后，系统自动按**日期**聚合所有菜的食材。

进入方式：
- 首页 → 点击「🛒 采购清单」
- 我的 → 点击「🛒 采购清单」

### 5.2 使用清单

清单会显示：

```
☐ 鸡蛋 × 9个       来自：西红柿炒鸡蛋、蛋炒饭
☐ 西红柿 × 4个     来自：西红柿炒鸡蛋
☑ 面粉 × 500克     来自：手擀面
```

**操作**：
- 🗓 **切换日期**：顶部可查看不同日期的清单
- ✅ **勾选已购**：点击食材项标记为已购买（会自动保存勾选状态）
- 📋 **复制清单**：点击底部「复制清单」按钮，清单文本复制到剪贴板，可粘贴到微信发给家人

### 5.3 清单示例（复制效果）

```
📋 采购清单 - 2025-07-04
共 3 道菜
━━━━━━━━━━━━━━

☐ 鸡蛋 × 9 个
   来自：西红柿炒鸡蛋、蛋炒饭

☐ 西红柿 × 4 个
   来自：西红柿炒鸡蛋

☐ 面粉 × 500 克
   来自：手擀面
```

## 7. 使用场景

### 场景一：妈妈做菜，家人点菜

```
👨 爸爸：打开小程序 → 菜谱 → 找到「红烧排骨」→ 点这道菜
                            → 明天午餐 → 备注「少放糖」→ 确认

👧 女儿：同上 → 点了「番茄炒蛋」→ 明天午餐

👩 妈妈：打开小程序 → 首页看到 2 个待确认订单
       → 逐一确认 → 去采购清单查看要买什么
       → 买菜 → 做菜 → 标记已完成
```

### 场景二：周末大餐计划

```
周四晚上：
  家人各自浏览菜谱，点上周末想吃的大菜
  
周五：
  厨师打开采购清单 → 选择周六的日期 → 复制清单 → 发到家庭群
  照着清单去超市采购

周六：
  按订单先后顺序做菜 → 每完成一道标记「已完成」
```

### 场景三：添加新菜谱

```
今天做了道新菜「宫保鸡丁」很好吃

1. 打开小程序 → 菜谱 → 点 + 号
2. 用手机拍张成品照当封面
3. 输入菜名、分类、简介
4. 添加食材：鸡胸肉、花生、干辣椒…
5. 添加步骤（每步可拍照）：
   - 步骤1：鸡胸肉切丁，加料酒生抽腌制
   - 步骤2：花生小火炒香
   - ……
6. 保存！现在家人就可以点这道菜了
```
